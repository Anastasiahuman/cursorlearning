# Руководство для команды - Проект йога-приложения

## Обзор проекта

Создаём мобильное приложение для практики йоги с 40 готовыми видео-уроками, структурированными программами и персонализацией.

**Ключевые направления:**
- Здоровье спины
- Развитие гибкости
- Работа с тазобедренными суставами
- Медитация и осознанность
- Релаксация и восстановление

GET  /api/programs/:id         # Детали программы
GET  /api/categories           # Категории

POST /api/users/progress       # Сохранение прогресса
GET  /api/users/progress       # Получение прогресса

### Схемы БД

**Таблица lessons:**
```sql
CREATE TABLE lessons (
  id VARCHAR(3) PRIMARY KEY,
  title VARCHAR(100) NOT NULL,
  category VARCHAR(50) NOT NULL,
  level INTEGER CHECK (level BETWEEN 1 AND 3),
  duration INTEGER NOT NULL,
  description TEXT,
  instructor VARCHAR(100),
  video_url TEXT NOT NULL,
  thumbnail_url TEXT NOT NULL,
  tags TEXT[],
  benefits TEXT[],
  contraindications TEXT[],
  equipment TEXT[],
  focus_areas TEXT[],
  intensity VARCHAR(20),
  style VARCHAR(50),
  is_premium BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

**Таблица programs:**
```sql
CREATE TABLE programs (
  id VARCHAR(20) PRIMARY KEY,
  title VARCHAR(100) NOT NULL,
  description TEXT,
  duration_weeks INTEGER,
  lessons TEXT[],  -- Array of lesson IDs
  goal TEXT,
  level INTEGER,
  category VARCHAR(50),
  is_premium BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW()
);
```

**Таблица user_progress:**
```sql
CREATE TABLE user_progress (
  user_id VARCHAR(50),
  lesson_id VARCHAR(3),
  completed BOOLEAN DEFAULT FALSE,
  completed_at TIMESTAMP,
  PRIMARY KEY (user_id, lesson_id),
  FOREIGN KEY (lesson_id) REFERENCES lessons(id)
);
```

### Целевая аудитория (4 сегмента)

**Сегмент 1: Офисные работники (25-45 лет)**
- Pain points: боли в спине, стресс, сидячая работа
- Мотивация: здоровье, снятие напряжения
- Каналы: LinkedIn, Facebook, контекстная реклама

**Сегмент 2: Начинающие практики (20-50 лет)**
- Pain points: не знают с чего начать, нет времени на студию
- Мотивация: освоить йогу, улучшить гибкость
- Каналы: Instagram, YouTube, блоги о здоровье

**Сегмент 3: Активные пользователи (25-55 лет)**
- Pain points: нужна растяжка после тренировок
- Мотивация: дополнить тренировки, восстановление
- Каналы: фитнес-сообщества, спортивные блоги

**Сегмент 4: Люди с ограничениями (30-60 лет)**
- Pain points: проблемы со здоровьем, нужен щадящий подход
- Мотивация: улучшить самочувствие, снять боль
- Каналы: медицинские форумы, рекомендации врачей
