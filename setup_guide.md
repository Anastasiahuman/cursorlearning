# Настройка Cursor: пошаговая инструкция

Инструкция для тех, кто никогда не работал с редакторами кода. Всё просто — справится любой.

---

## Шаг 1. Скачай и установи Cursor

1. Перейди на сайт **cursor.com**
2. Нажми **Download** — сайт сам определит твою систему (Mac / Windows)
3. Установи как обычное приложение:
   - **Mac**: открой скачанный `.dmg`, перетащи Cursor в папку Applications
   - **Windows**: запусти `.exe` и следуй установщику

## Шаг 2. Создай аккаунт

1. При первом запуске Cursor предложит войти или зарегистрироваться
2. Можно через Google-аккаунт — так быстрее всего
3. На бесплатном тарифе есть 2 недели Pro — этого хватит для тренинга

## Шаг 3. Выбери модель (важно!)

1. В нижнем правом углу рядом с чатом будет выбор модели
2. Выбери **Claude Sonnet** или **claude-3.5-sonnet** — она лучше всего справляется с задачами менеджера
3. Если доступен **Claude Sonnet 4** — выбирай его, это самая свежая версия

## Шаг 4. Открой папку с кейсами

1. **File → Open Folder** (или `Cmd+O` на Mac / `Ctrl+O` на Windows)
2. Выбери папку `cursorlearning` (этот репозиторий)
3. Слева появится дерево файлов — папки `cases/` и `data/`

## Шаг 5. Открой чат с AI

Есть два способа общения с AI в Cursor:

### Способ A: Чат (основной)
- Нажми `Cmd+L` (Mac) или `Ctrl+L` (Windows)
- Откроется панель чата справа
- Пиши промпт текстом, как в ChatGPT
- Cursor видит все файлы в открытой папке и может их читать и создавать

### Способ B: Composer (для создания файлов)
- Нажми `Cmd+I` (Mac) или `Ctrl+I` (Windows)
- Composer лучше подходит, когда нужно создать или отредактировать несколько файлов сразу
- Работает так же — пишешь что нужно, AI делает

## Шаг 6. Выполни первый кейс

1. Открой файл `cases/case_1_prd.md` — это кейс по созданию PRD
2. Скопируй промпт из раздела «Промпт для Cursor»
3. Вставь в чат (`Cmd+L`) и нажми Enter
4. Cursor прочитает бриф, контекст и шаблон, и создаст готовый PRD
5. Открой созданный файл `prd_stories.md` — готово!

> Для WOW-эффекта попробуй **Кейс 2** — Cursor за 3 минуты соберёт интерактивную презентацию с графиками из JSON. Открой `cases/case_2_presentation.md`.

| Открыть терминал | `` Ctrl+` `` | `` Ctrl+` `` |

## Частые вопросы

**Cursor — это бесплатно?**
Есть бесплатный тариф с ограничениями. Для тренинга достаточно триала Pro (2 недели бесплатно).

**Нужно ли уметь программировать?**
Нет. Ты пишешь задачу обычным языком, Cursor пишет код за тебя. Ты только проверяешь результат.

**Что если Cursor сделал не то, что нужно?**
Напиши в чат что не так. Например: «Измени цвет графиков на синий» или «Добавь ещё колонку с процентом от общей выручки». Cursor поправит.

**Как отменить изменения?**
`Cmd+Z` (Mac) / `Ctrl+Z` (Windows) — как в любом редакторе.

**Cursor видит мои файлы?**
Cursor видит только файлы в открытой папке. Он не имеет доступа к остальному компьютеру.
