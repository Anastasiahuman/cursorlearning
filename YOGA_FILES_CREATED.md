## Скрипты (Python)

### 19. yoga-content/scripts/import_lessons.py
**Описание:** Скрипт для импорта и структурирования уроков

**Функционал:**
- Импорт видео из исходной папки
- Создание структуры папок для каждого урока
- Генерация metadata.json
- Создание description.md
- Валидация по JSON схеме
- Массовый импорт из конфигурации
- Генерация индексного файла

**Размер:** ~8 KB
**Зависимости:** jsonschema

**Использование:**
```bash
python import_lessons.py --source ../source_videos --output ./content/lessons --config ./config/lessons_config.json
```

### 20. yoga-content/scripts/validate_content.py
**Описание:** Скрипт для валидации контента

**Функционал:**
- Валидация уроков по JSON схеме
- Проверка наличия файлов (видео, превью, описания)
- Валидация программ
- Проверка ссылок между уроками и программами
- Проверка категорий
- Генерация отчёта о валидации

**Размер:** ~10 KB
**Зависимости:** jsonschema

**Использование:**
```bash
python validate_content.py --content ./content --schemas ./schemas
```
