# Кейс 2: Оптимизация воронки — «Где мы теряем пользователей?»

## Контекст для участников
Вы — продакт-менеджер, отвечающий за онбординг. Команда роста поставила OKR: **увеличить конверсию из установки в завершённый онбординг с 25.7% до 35% за квартал**. Нужно найти узкие места и предложить план действий.

## Что делаем в Cursor (пошагово)

### Шаг 1. Анализ воронки
Открываем чат и пишем:

```
Проанализируй funnels/onboarding_funnel.json.
Найди самые большие потери в воронке.
Где наибольший drop-off? Сравни декабрь и январь.
Визуализируй в виде таблицы.
```

### Шаг 2. AI находит bottleneck
Ожидаемый ответ:
- **install → signup**: потеря ~29% (самый большой drop абсолютных чисел)
- **signup → profile_setup**: потеря 18% (форма профиля — барьер)
- **profile → family_create**: потеря 27% (не все хотят создавать семью сразу)
- Январь лучше декабря: signup конверсия 71.2% vs 68.5%, полный онбординг 25.7% vs 20.9%

### Шаг 3. Просим предложить решения
```
Исходя из данных воронки и product/app_concept.json,
предложи 5 конкретных гипотез для увеличения конверсии
онбординга до 35%. Для каждой укажи:
- Что меняем
- На какой шаг воронки влияет
- Ожидаемый эффект
- Сложность реализации (S/M/L)
```

### Шаг 4. Связываем с A/B-тестами
```
Посмотри experiments/ab_tests.json — тест EXP-042
про видео в онбординге. Как его результаты помогают
достичь цели 35%? Посчитай, хватит ли одного этого
теста или нужны дополнительные изменения.
```

### Шаг 5. Формируем roadmap
```
Составь план на квартал из 3-4 экспериментов
для достижения 35% конверсии онбординга.
Для каждого: гипотеза, метрика, целевой lift, приоритет.
Формат — таблица.
```

## Чему учит этот кейс
- **Работа с воронками**: как читать конверсии, находить bottleneck
- **Data-driven решения**: гипотезы строятся на данных, а не на интуиции
- **Связка данных**: AI сам подтягивает результаты прошлых тестов
- **Планирование**: от проблемы → к гипотезам → к roadmap экспериментов

## Промпты для практики участников
1. `Сравни конверсию онбординга по каналам (organic vs paid vs referral). Откуда приходят лучшие пользователи?`
2. `Если бы у нас был бюджет только на 1 изменение в онбординге, что бы ты порекомендовал и почему?`
3. `Посчитай, сколько дополнительных paying users мы получим, если увеличим конверсию онбординга с 25.7% до 35%`
