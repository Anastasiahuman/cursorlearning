# Бонус: Экспертная карусель для Instagram из статьи

## Ситуация
У тебя есть статья / текст / заметки, и нужно сделать стильную карусель для Instagram — как в экспертных блогах. Обычно это занимает 2-3 часа в Figma или Canva. Cursor сделает это за 3 минуты.

## Данные
- `data/article_loyalty_trends.md` — статья о трендах программ лояльности

## Промпт для Cursor

```
Прочитай data/article_loyalty_trends.md — это статья про тренды программ лояльности.

Сделай HTML-файл carousel.html — набор слайдов для Instagram-карусели в экспертном стиле.

Дизайн:
- Размер каждого слайда: 1080x1350px (формат 4:5 для Instagram)
- Чередование тёмных (фон #1a1a1a) и светлых (фон #f5f0eb) слайдов
- Типографика: заголовки — крупный serif шрифт (Georgia или Playfair Display из Google Fonts), текст — sans-serif
- Акцентный цвет: #c0392b (красный) для ключевых слов и цифр
- Категория/тема слайда — мелким капслоком сверху (ПРОБЛЕМА, ДАННЫЕ, РЕШЕНИЕ и т.д.)
- Нумерация слайдов в правом верхнем углу на светлых слайдах
- Крупные цифры/статистика — вынести в карточки с рамкой
- Цитаты — выделить в отдельный блок внизу слайда, курсивом
- Буллеты с эмодзи-иконками

Структура слайдов:
1. Обложка (тёмный): яркий заголовок + подзаголовок + «ЛИСТАЙ →»
2. Масштаб проблемы (тёмный): 4 ключевых числа в карточках
3. Что не работает (светлый): 3 пункта с иконками
4. Эмоциональная лояльность (тёмный): пример Nike
5. Персонализация (светлый): пример Kroger + цифры
6. Геймификация (тёмный): пример Duolingo
7. Ключевые цифры (светлый): 4 метрики крупно
8. Что делать (светлый): чек-лист с галочками
9. CTA (тёмный): призыв к действию + «сохрани / отправь другу»

Каждый слайд — отдельный div с классом .slide. Добавь кнопки навигации ← → для просмотра.
На каждом слайде не больше 3-4 элементов — всё должно быть читаемо с телефона.
```

## Как сделать скриншоты для Instagram

После генерации:
1. Открой `carousel.html` в браузере
2. Для каждого слайда: правый клик → «Inspect» → выбери `.slide` → правый клик на элементе → «Capture node screenshot»
3. Или используй расширение GoFullPage / FireShot
4. Загружай в Instagram как карусель

Альтернативный промпт для автоматизации:
```
Добавь кнопку «Скачать все слайды как PNG» используя html2canvas из CDN
```

## Что демонстрирует
- Cursor создаёт дизайн-контент, не только код и документы
- Из текста за 3 минуты — готовый визуальный контент для соцсетей
- Можно итерировать: «Измени цвета на фирменные», «Добавь лого», «Сделай в стиле Тинькофф-журнала»
- Экономит 2-3 часа работы дизайнера или время в Canva

## Время: 3-4 минуты
