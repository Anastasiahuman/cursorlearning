# Кейс 2: Сборка презентации по метрикам (WOW-демо)

## Ситуация
Завтра ежемесячный стендап с руководством. Нужна красивая презентация с метриками за январь. Данные есть в JSON, времени делать слайды в PowerPoint — нет.

## Данные
- `data/app_context.md` — контекст приложения
- `data/metrics_jan2026.json` — все метрики за январь 2026

## Промпт для Cursor

```
Прочитай:
- data/app_context.md — контекст приложения LoyalBox
- data/metrics_jan2026.json — метрики за январь 2026

Сделай HTML-презентацию (файл presentation.html) для ежемесячного отчёта руководству:

Слайды:
1. Титульный — «LoyalBox: итоги января 2026»
2. Ключевые метрики — MAU, DAU, Retention, NPS (крупные цифры + изменение vs декабрь)
3. Рост по месяцам — график тренда MAU/DAU с сентября по январь (Chart.js)
4. Программа лояльности — баллы начислено/списано, средний чек, разбивка по уровням
5. Push-уведомления — open rate, лучшие типы пушей
6. Реферальная программа — конверсия, влияние на retention
7. NPS и обратная связь — оценка + топ жалоб
8. Фокус на февраль — что планируем (возьми из контекста)

Требования:
- Навигация стрелками ← → между слайдами
- Графики через Chart.js (CDN)
- Красивый дизайн с градиентами и карточками
- Цифры роста зелёным, падения красным
- Формат 16:9, чтобы можно было показать на проекторе
```

## Что демонстрирует (WOW-эффект)
- Из сырого JSON за 2 минуты — готовая презентация с графиками
- Интерактивные слайды, работающие в браузере
- Выглядит как работа дизайнера, а не шаблон PowerPoint
- Можно итерировать: «добавь слайд про...», «измени цвета на...»

## Время: 3-4 минуты

## Бонус: итерация

```
Добавь анимацию появления элементов на каждом слайде и индикатор текущего слайда внизу
```
