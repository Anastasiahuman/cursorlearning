# Кейс 3: Работа с таблицами и данными

## Ситуация
Маркетинг прислал выгрузку по рекламным кампаниям за январь. Нужно привести данные в порядок: посчитать ROI по каждой кампании, найти лучшие и худшие, подготовить сводку для встречи с маркетингом.

## Данные
- `data/campaigns_table.csv` — 10 кампаний: бюджеты, показы, клики, конверсии, выручка

Проблема: в таблице есть пропуски (не у всех кампаний заполнена выручка), разные типы кампаний, и нужно посчитать производные метрики.

## Промпт для Cursor

```
Посмотри файл data/campaigns_table.csv — это данные по маркетинговым кампаниям приложения LoyalBox за январь.

Сделай HTML-дашборд (файл campaigns_dashboard.html):

1. Сводная таблица со всеми кампаниями + рассчитанные метрики:
   - CTR (clicks / impressions)
   - CR (conversions / clicks)
   - CPA (budget / conversions)
   - ROI ((revenue - budget) / budget), где есть данные по выручке
2. Подсветить зелёным топ-3 кампании по ROI, красным — худшие
3. Отдельный блок: сравнение типов кампаний (Push vs Акция vs Реферал и т.д.) — средний CTR, CR, CPA по типу
4. Рекомендации: какие кампании масштабировать, какие остановить

Данные, где нет выручки — пометить «нет данных» и не считать ROI.
Дизайн — чистый, табличный, как дашборд аналитика.
```

## Что демонстрирует
- Cursor работает с CSV, считает производные метрики
- Обрабатывает пропуски в данных корректно
- Делает сортировку и визуальное выделение
- Генерирует аналитические рекомендации на основе данных
