## Чему учит этот кейс
- **Статистическая грамотность**: p-value, confidence level, sample size — что значат для решения
- **Ship/Kill framework**: как принимать решение по тесту, не только по primary метрике
- **Риск-менеджмент**: как учитывать побочные эффекты (coin inflation)
- **Коммуникация**: как быстро подготовить питч по результатам теста
- **Peeking problem**: почему нельзя принимать решение по промежуточным данным
